# Mini-Projects
Small standalone but cool programs
1) A-Maze-Ing!.sb3: Generates a random maze and uses Djikstra's algorithm to solve it.
2) FSSE: Visualizes Fraunhofer Single Slit Diffraction pattern with different colours and parameters.
3) Queen.py: Finds all possible ways to place n Queens in a nxn chessboard such that none of them attack each other.
4) Waves.sb3: Simulates 2 transverse waves with variable time period, wavelength and amplitude, and their superposition.
5) YDSE.sb3: Visualizes Young's Double Slit Experiment Interference Pattern with different colours and parameters.
